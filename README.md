## Getting Started

### Importing the library

To use the library, first import it:

In Node:
```javascript
const { compose } = require('baccano')
```

As ES Module:
```javascript
import { compose } from 'baccano'
```

On the browser:
```html
<script src="https://cdn.jsdelivr.net/npm/baccano/baccano.min.js"></script>
<script>
    const { compose } = Baccano
</script>
```
